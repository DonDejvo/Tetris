# Tetris

Play in browser:

[Tetris](https://dondejvo.github.io/Tetris/)

This was compiled by emscripten

## How to run

Prerequisities:

SDL2, SDL2_ttf and SDL2_image

Ubuntu:

```bash
sudo apt update
sudo apt install libsdl2-dev libsdl2-ttf-dev libsdl2-image-dev
```

Then build with cmake.

Targets:

- Tetris
- TetrisTests
